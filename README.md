# Deno Oak + React Router + GraphQL Demo

Full-stack Deno app with React Router with RSC, GraphQL (Pothos/GQty), Drizzle (Postgres), Redis sessions.

[CodeSandbox](https://codesandbox.io/p/github/Nyoxis/oak-reactRouter-graphqQL)

### Tasks

| Task               | Purpose                                 |
| ------------------ | --------------------------------------- |
| deno task push     | Push pothos schema to DB                |
| deno task generate | Start server and generate gqty schema   |
| deno task build    | Build vite app                          |
| deno task dev      | Run hot reload server and frontend      |
| deno task start    | Run server serving backend and frontend |
